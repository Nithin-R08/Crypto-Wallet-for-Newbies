# 🔐 CipherSafe – Crypto Wallet for Newbies

CipherSafe is a beginner-friendly, secure, and modern crypto wallet interface designed to help first-time users understand and manage cryptocurrencies easily.  
The platform provides an intuitive UI, guided steps for sending/receiving crypto, and a clean Web3-inspired design that builds trust and simplicity for new users.

---

## 📌 Project Overview

CipherSafe simplifies crypto usage by offering:

- 🔑 Secure PIN-based wallet unlocking  
- 🔐 Biometric authentication option  
- 💼 Wallet balance overview  
- 💸 Send & receive crypto with guided steps  
- 🧾 Detailed transaction history  
- 🎨 Modern, calming, gradient UI for beginners  
- 📱 Responsive design for mobile, tablet & desktop  

Built using **React / Next.js**, **Tailwind CSS**, and **clean component architecture**, the platform ensures smooth onboarding and ease of use.

---

## 💡 Key Highlights

###  1. Wallet Unlock Screen  
New users can easily unlock their wallet using:
- A simple 4-digit PIN  
- OR biometric authentication (fingerprint)

This makes onboarding extremely beginner-friendly.

---

###  2. Beginner-Friendly Dashboard  
The dashboard displays:
- BTC / ETH / USDT balances  
- USD value equivalent  
- Send & Receive quick buttons  
- Clean cards with subtle animations  

Perfect for helping newbies understand their assets at a glance.

---

###  3. Step-by-Step Crypto Sending  
Sending crypto is broken into easy steps:

1️⃣ Enter Recipient Address  
2️⃣ Enter Amount + Select Crypto  
3️⃣ Review Details  
4️⃣ Confirm & Send  

This reduces user error and builds confidence for first-time crypto users.

---

###  4. Transaction History  
Shows:
- Asset type (BTC / ETH / USDT)  
- Amount  
- Status (Success / Pending / Failed)  
- Timestamp  

New users clearly understand what happened to their transactions.

---

###  5. Clean, Modern UI  
- Smooth gradient backgrounds  
- Minimal neon highlights  
- Dark mode interface  
- Mobile-friendly layout  
- Simple, trust-focused design  

---

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository
```bash
git clone https://github.com/your-username/Crypto-Wallet-For-Newbies.git
cd Crypto-Wallet-For-Newbies
```

### 2️⃣ Install Dependencies
```bash
npm install
```

### 3️⃣ Start Development Server
```bash
npm run dev
```

Open the app in your browser:  
👉 `http://localhost:3000`
